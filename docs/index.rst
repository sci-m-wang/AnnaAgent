AnnaAgent Documentation
=======================

AnnaAgent is a dynamic evolution agent system with multi-session memory for
realistic seeker simulation in psychological counseling research. This site
contains bilingual documentation for installation, workspace setup, CLI usage,
model deployment, memory indexing, and project maintenance.

AnnaAgent 是面向心理咨询研究的动态演化来访者模拟系统，支持跨疗程记忆、情绪调制、主诉链生成和命令行实验流程。本文档提供中英双语说明，帮助读者从安装到运行、从基础模型到 SFT/vLLM 部署逐步上手。

.. note::

   The public repository is intended for method demonstration, reproducible
   experiments, and downstream research tooling. It does not include restricted
   raw counseling records.
